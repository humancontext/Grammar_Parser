GENERAL USAGE NOTES
--------------------------------------------------------------------------------------------------
- To test the Context-Free Grammar parser, use
	python cfg_parser.py

- To test the Unification Grammar parser, use
	python unig_parser.py

- You are allowed to test your own sentences with the parser generated from the rules,
	or revisit the parsing tree of the build-in sentences by inputing the index.

- Use ctrl+c or q to quit the program.

- The code is to be compiled python2.7.

- Required packages:  nltk, sys.
